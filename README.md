## Getting Started

1. Clone the repository
```bash
git clone https://github.com/yourusername/data-science-projects.git
cd data-science-projects
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook
```bash
jupyter notebook
```
